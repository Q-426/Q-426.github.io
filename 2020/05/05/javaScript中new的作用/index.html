javaScript中new的作用

发表于 2020-05-05 | 分类于 js基础 ， 前端基础

1. 创建一个全新的对象

相当于var obj = {};这样，创建了一个Object实例，在内存中开辟了一块空间。

2. 这个新对象会被执行[[prototype]]连接

首先说这个 [[Prototype]], 我们都知道原型链这个东西, 原型链的寻找过程其实不是 xxx.prototype 指向的那个东西依次向上找, 而是一个内部的指针 [[Prototype]] 的依次向上寻找, 也就是各大浏览器里面实现的 proto 这个东西

新对象被执行[[prototype]]连接，其实要做的事就是

3. 新对象绑定到函数调用的this

在new后面跟着的构造函数，在这条语句执行之前，内部的this指向全部都被指定为新生成的这个obj

4. 如果函数没有返回其他对象, 那么 new 表达式中的函数调用会自动返回这个新对象

这句话的意思, 就是说, 我在构造函数的最后 return {obj : 1} 出来, 最后生成的不会是那个实例, 而会是 {obj : 1}

需要注意的一点就是, 这里所说的没有返回其他对象, 必须是对象, 加入你是 return 1, 还是会生成实例而不是那个 1
